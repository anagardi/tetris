Pajitnov's original version for the Electronika 60 computer used green brackets to represent blocks. Versions of Tetris on the original Game Boy/Game Boy Color and on most dedicated handheld games use monochrome or grayscale graphics, but most popular versions use a separate color for each distinct shape. Prior to The Tetris Company's standardization in the early 2000s (decade), those colors varied widely from implementation to implementation.
